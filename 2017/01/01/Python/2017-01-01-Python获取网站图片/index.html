安装Beautiful Soup

由于本机使用的是Python3.5, 但是mac自带的是Python2.6, 所以直接执行easy_install会安装2.6版本的beautifulsoup, 所以需要使用命令easy_install-3.5, 可以到/Library/Frameworks/Python.framework/Versions/3.5/bin 下查看命令
