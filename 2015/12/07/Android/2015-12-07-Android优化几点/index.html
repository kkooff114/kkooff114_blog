如何避免 Activity 泄漏?

  • 移除掉所有的静态引用。
  • 考虑用 EventBus 来解耦 Listener。
  • 记着在不需要的时候，解除 Listener 的绑定。
  • 尽量用静态内部类。
  • 做 Code Review。个人经验：Code Review 能很早的发现内存泄漏。
  • 了解你程序的结构。
  • 用类似 MAT，Eclipse Analyzer，LeakCanary 这样的工具分析内存。
  • 在 Callback 里打印 Log。

滑动 不卡顿

  • UI 线程只用作 UI 渲染

  • 不要在 UI 线程做下面的事情：

    载入图片
    网络请求
    解析 JSON
    读取数据库
